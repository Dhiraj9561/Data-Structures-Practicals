# 📌 About the Project

This repository contains implementations of **Data Structures and Algorithms (DSA)** using **C and C++**.

The purpose of this repository is to practice and understand the **fundamental concepts of data structures**, which are essential for **problem-solving, competitive programming, and technical interviews**.

All programs demonstrate basic DSA concepts such as **Arrays, Queues, Linked Lists, and Searching Algorithms**.



# 📚 Topics Covered

| Topic                  | Description                            |
| ---------------------- | -------------------------------------- |
| 📦 Arrays              | Basic array implementation             |
| 🔢 2D Arrays           | Matrix representation                  |
| 📥 Queue (Array)       | Queue implementation using arrays      |
| 🔗 Queue (Linked List) | Queue implementation using linked list |
| 🔗 Linked List         | Single linked list operations          |
| 🔎 Searching           | Linear search & binary search          |

# ▶️ How to Run the Programs

### 1️⃣ Compile C Program

```bash
gcc filename.c
```

### 2️⃣ Compile C++ Program

```bash
g++ filename.cpp
```

### 3️⃣ Run the Program

```
a.exe
```

or

```
./a.exe
```
